1. опубликовать бинарники: build -> publish -> file system -> ok
2. установить iis: панель управления -> программы -> включение\отключение компонентов -> службы iis -> ок
3. установить в него asp.net: win+r -> 'C:\Windows\Microsoft.NET\Framework64\v4.0.30319\aspnet_regi is.exe -i'
4. в iis: пулы приложений -> добавить пул -> .net 4.0, встроенный конвейер -> сохранить
5. в iis: сайты -> добавить веб-сайт -> указать созданный пул, физический путь, порт -> ок
6. в зависимости от настроек windows возможно потребуется дать доступ для пользователей IIS_IUSRS и\или NETWORK SERVICE к папке сайта
